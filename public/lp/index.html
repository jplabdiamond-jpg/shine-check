PROBLEM

伝票を手書きしていませんか？

現場あるある、こんなトラブルが毎晩起きていませんか？

ドリンク数の記入ミス

「シャンパン2本頼んだのに1本しか書いてない」「ソフトドリンクとアルコールを混同した」—手書きだからこそ起きる計上漏れ。

会計時の計算ミス

税率・サービス料・指名料…複数の料金を電卓で計算するうちにミス発生。お客様を待たせてしまい、クレームの原因に。

文字が読めない・紛失する

忙しい夜に書いたメモは翌朝解読不能。伝票そのものが行方不明になり、売上集計が正確にできない。

在席時間の管理が曖昧

「何時から座ってる？」「延長は何回した？」—セット・延長料金の請求を口頭で管理すると、いつかトラブルになる。

売上集計に毎朝1〜2時間

手書き伝票を積み重ねて電卓で合計を出す作業。閉店後の疲れた状態でやるから計算ミスも多発する。

キャストのバック計算が大変

指名・同伴・飲みバック・時給…それぞれ料率が違うのに、全部手計算。キャストへの支払いトラブルの温床に。

そこで —

Shine Check の出番です

SOLUTION

水商売の現場に
特化したクラウド伝票管理ツール

伝票・売上・キャスト管理を、スマホ1台で完結。

複雑な料金計算もセット/延長タイマーも全自動。
開店から閉店まで、現場の流れをそのままシステム化。
無料プランでも主要機能をフル活用できます。

FEATURE 01

ダッシュボード

今夜の売上・卓・伝票を
ひと目で把握

開くだけで当日の売上・伝票枚数・使用卓数がリアルタイム表示。営業終了ボタン1つで未会計伝票を一括処理し、日次売上を自動保存します。

  • 確定売上 / 未確定売上を分離表示
  • 営業中の伝票リストをダッシュボードに常時表示
  • 卓状況サムネイルで使用中・空席を一覧確認
  • 「営業終了」で全open伝票を一括自動会計
  • 無料プランの伝票上限を警告バナーで通知
FEATURE 02

卓管理 & タイマー

「延長まであと何分？」を
自動カウントダウン

各卓にリアルタイムタイマーを設置。セット時間が近づくと枠が赤く点滅してアラート。延長追加でタイマーも自動リセット。見逃し・トラブルゼロへ。

  • 1秒単位のリアルタイムカウントダウン
  • セット終了が近づくと点滅アラート（BlinkTimer）
  • デフォルト60分・90分・120分からワンタップ選択
  • 空席 / 使用中 / 予約 のステータスを即切替
  • 卓の追加・名前変更・アーカイブに対応
+
Remote Access

卓の状況やタイマー状況、その卓のリアルタイム売上も
全てログイン端末で遠隔で確認できます！

FEATURE 03

伝票管理

作成から会計・領収書まで
スマホ1台で完結

卓を選んでセット料金を追加するだけで伝票が完成。ドリンク・指名料・延長料も明細追加でき、会計時に税・サービス料を自動計算。領収書もワンタップ表示。

  • セット料金・延長・指名・同伴料を個別追加
  • 消費税 / サービス料を設定値で自動計算
  • 現金 / カード / その他の支払い方法に対応
  • 領収書モーダルをワンタップ表示
  • 無料プランは1日15枚まで（プレミアム無制限）
FEATURE 04

商品管理

メニューを登録して
伝票入力を爆速に

ドリンク・ボトル・フードをカテゴリ別に登録。伝票作成時にセレクタから選ぶだけで素早く追加でき、手入力ミスも防ぎます。

  • カテゴリ別（ドリンク / フード / ボトル / その他）に整理
  • よく使う商品はお気に入り★でトップ表示
  • 商品追加・編集・削除がシンプル操作で完結
  • 伝票画面からセレクタで即選択・追加
FEATURE 05

キャスト管理プレミアム

バック計算・時給を
全自動で算出

飲みバック・指名バック・同伴・延長バック率を個別設定。出退勤打刻から実働時間・時給・人件費サマリーまで自動計算し、給与トラブルを根絶します。

  • 本名・源氏名を分けて管理
  • 飲み / 指名 / 同伴 / 延長バック率を個別設定
  • 出勤打刻（時間指定可）・退勤打刻に対応
  • 実働時間を自動計算（15分以下切捨・16分以上繰上）
  • 本日・7日・14日・30日の期間集計
+
Auto Calculation

キャストの時給やバック率を設定しておく事で、
営業終了時には自動的に全キャスト給料が計算され表示されます！

FEATURE 06

顧客管理プレミアム

常連・VIP客の情報を
スタッフ全員で共有

好み・NGメモ・来店履歴をアプリで一元管理。次回来店時に前回情報を即確認でき、スタッフが替わっても質の高いおもてなしを提供できます。

  • VIPフラグ・誕生日・電話番号・かな管理
  • 好み・NGメモをスタッフ全員で共有
  • 来店履歴を伝票と自動連携して蓄積
  • 顧客検索で名前・かなから即ヒット
+
Staff Sharing

顧客情報を細かく記載することで、どのスタッフが対応しても
スタッフの端末で情報をいつでも確認できます！

FEATURE 07

売上分析プレミアム

売上グラフ・商品ランキングで
データ経営を実現

日別売上グラフと商品別売上ランキングで、売れ筋・稼ぎ頭が一目瞭然。AIインサイトが最も売れた商品を自動でハイライトします。

  • 日別売上推移グラフ（折れ線）
  • 期間切替：7日 / 14日 / 30日
  • 商品別売上ランキング自動集計
  • AIインサイトで注目商品をハイライト表示
  • 総売上・伝票枚数・客単価をサマリー表示
FEATURE 08

設定

店舗の料金体系を1回設定すれば
あとは全自動

消費税・サービス料・セット/延長のデフォルト料金を設定しておくと、伝票作成時に自動反映。設定変更はいつでも即時反映されます。

  • 消費税・サービス料を % / 固定額で設定
  • セット・延長のデフォルト料金と分数を設定
  • キャスト時給計算単位（30分 / 1時間）を選択
  • 店舗名変更・プラン確認もここから
